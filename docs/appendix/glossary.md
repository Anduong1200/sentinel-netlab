# Glossary

- **BSSID**: Basic Service Set Identifier. Uniquely identifies an AP (MAC address).
- **Beacon Frame**: Management frame broadcast by APs to announce their presence.
- **Deauthentication**: Management frame used to disconnect a client (often spoofed in DoS attacks).
- **Evil Twin**: A rogue AP impersonating a legitimate network by cloning its SSID.
- **HMAC**: Hash-Based Message Authentication Code. Used to sign telemetry.
- **Monitor Mode**: Network card mode allowing capture of all air traffic, not just frames addressed to the device.
- **RSSI**: Received Signal Strength Indicator.
- **Sentinel**: The sensor agent running on the edge device.
